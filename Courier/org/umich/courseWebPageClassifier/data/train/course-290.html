Course Description
  Top

The course will follow the textbook with some extra material on Zeta-functions and, if time permits, global class field theory at the end. After an introduction on diophantine equations and reciprocity laws we start with the systematic development: Rings of integers  in Number fields, the discriminant, unique factorization, splitting of primes, the ideal class group and the unit group, the Dedekind Zeta function and the analytic class number formula, some aspects of global class field theory.
