COURSE DESCRIPTION

The course will give an introduction to global class field theory. We will first discuss the classical approach using the analytic theory of \(L\)-functions. A good and complete treatment is in Janusz book below. We will then treat what is now the standard approach, centering around the concept of a class formation, which doesn't use analytic tools. The standard reference for this is Tate's article in the volume by Cassels-Fröhlich. Time permitting we will discuss the duality theorems of Tate-Poitou and Artin-Verdier.
